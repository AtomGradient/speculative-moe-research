Key Findings核心发现

SD Works for MoESD 对 MoE 有效

Despite very low acceptance rates (<4%), speculative decoding provides 1.18–1.30× speedup on MoE models through batch verification efficiency. 尽管接受率极低（<4%），推测解码通过批量验证效率仍能为 MoE 模型提供 1.18–1.30× 加速。

Total Params Matter总参数量决定效果

SD speedup scales with total parameter footprint (memory bandwidth), not active parameters. MoE (35B total, 3B active) behaves like a 4–9B dense model. SD 加速与总参数量（内存带宽）成正比，而非活跃参数量。MoE（35B 总量，3B 活跃）表现类似 4-9B 的 Dense 模型。

Smaller Draft = Better更小的 Draft 更好

The 0.8B draft outperforms the 2B draft — lower compute overhead preserves more batch verification benefit. 0.8B 草稿模型优于 2B — 更低的计算开销保留了更多批量验证收益。

Larger γ = More Speedup更大的 γ = 更大加速

Counterintuitively, γ=16 gives the best speedup (1.30×) despite lowest acceptance (0.2%), because batch verification cost grows sub-linearly. 反直觉地，γ=16 提供最佳加速（1.30×），尽管接受率最低（0.2%），因为批量验证成本增长低于线性。
